# 🎮 Pokédex com API Pokémon

## 📖 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **Pokédex interativa** utilizando uma API pública de Pokémon para consultar informações em tempo real.

A aplicação permite pesquisar um Pokémon pelo nome ou número da Pokédex, exibindo dados como imagem, nome, identificação, tipos e outras informações disponíveis na API.

O projeto foi desenvolvido com o objetivo de praticar o consumo de APIs e a manipulação dinâmica de dados utilizando JavaScript.

---

## 🎯 Objetivos

* Consumir dados de uma API externa.
* Realizar requisições HTTP utilizando JavaScript.
* Trabalhar com dados em formato JSON.
* Atualizar a interface dinamicamente.
* Desenvolver aplicações interativas.
* Aprender a integrar Front-End com serviços externos.
